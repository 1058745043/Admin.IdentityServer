### 项目下载后，直接编译运行项目
```
Ctrl + F5
```

#### Tips：



```
如果你想换数据库，修改项目下的 appsettings.json 文件

  //数据库类型
  "type": 0,
  //连接字符串
  "connectionString": ""

```
